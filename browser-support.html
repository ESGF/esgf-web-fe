Web Front-End: Browser Support

The web front end makes use of newer browser technologies to provide the user with a rich, interactive experience. We make use of HTML5 features that are supported across many modern browsers.

    Supported Browsers
  • Mozilla Firefox
  • Apple Safari
  • Google Chrome
We also currently do NOT yet support IE browsers. This will be addressed in future releases!
